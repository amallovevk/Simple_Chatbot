# 🤖 Simple Keyword-Based Chatbot in Python

A beginner-friendly chatbot built using Python that responds to user inputs based on predefined keywords. This project demonstrates the fundamentals of Natural Language Processing (NLP) and conversational logic using a simple rule-based approach.

---

## 📌 Overview
This chatbot uses a keyword-matching technique to understand user input and provide relevant responses. It is designed as a basic introduction to chatbot development and can be extended into more advanced AI-based conversational systems.

---

## 🧠 Features
- Keyword-based response system  
- Interactive command-line interface  
- Simple and easy-to-understand logic  
- Beginner-friendly implementation  
- Easily extendable with more keywords and responses  

---

## ⚙️ Tech Stack
- Language: Python  
- Concepts: Basic NLP, String Matching  

---

## 🔍 How It Works
1. User enters a message  
2. Input is converted to lowercase  
3. Program checks for matching keywords  
4. If a keyword is found → corresponding response is returned  
5. If no match → default fallback response is shown
